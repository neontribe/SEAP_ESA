Your Data

This site does not send any of your data anywhere. It does collect some anonymous information such as how many people have used the tool so we can make improvements.

However, it does save some data just on the computer you’re using in order to work properly. If this is not your computer, tablet or phone you can delete all this data when you are done using the site. This way, no one else will see it.
